# Weekly Review SOP — 30 Minutes

Every Sunday evening. Inputs are pre-generated by agents overnight Saturday.

---

## 1. Finance Snapshot (5 min)

Open the latest row in `weekly_reports` (or the dashboard link from the Finance Agent email).

Check:
- [ ] Total revenue vs. last week
- [ ] Net revenue after fees
- [ ] Top 3 products by revenue
- [ ] Any anomalies flagged by the Finance Agent

Action: None usually required. Note trends.

---

## 2. Research Queue (10 min)

Open Supabase → `opportunities` table filtered to `status = pending`.

For each opportunity:
- Read the `title`, `rationale`, and `score`
- Decide: **approve** or **reject**
- Update `status` and add optional `notes`

Approving triggers the Creation Agent automatically (via Supabase webhook → n8n).

Target: approve 1–3 opportunities per week.

---

## 3. Listing Review (10 min)

Open Supabase → `listings` table filtered to `status = draft`.

For each draft listing:
- Review `title`, `description`, `tags`, `price_usd`
- Edit directly in Supabase or note changes for the Listing Agent to revise
- Update `status` to `published` to trigger auto-publish (Phase 3)

---

## 4. Marketing Queue (5 min)

Check the Marketing Agent output (Supabase or email digest):
- Review 3–5 social captions / Pinterest pin descriptions
- Approve or flag for revision
- Schedule posts (manual or via Buffer/Later in Phase 4)

---

## Done. See you next Sunday.
